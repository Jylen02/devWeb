Pour le bon fonctionnement du site, il faut importer dans un serveur sql tel que PhPAdmin la base de donnée "website_database.sql" présent 
dans le dossier "sql".

Pour lancer le site, il faut taper dans une barre d'adresse d'un navigateur l'URL suivante : localhost/DevWeb/php/accueil/home.php
Sur Windows, lancer Wampserver pour pouvoir interpreter les fichiers.php.
